# Offer Matcher

Локальный одностраничный инструмент для:

- вставки HTML страницы с вакансиями;
- выборки нужных блоков через CSS или XPath;
- просмотра извлеченного текста;
- отправки этого текста напрямую в OpenAI API для первичной оценки соответствия профилю.

## Запуск

Самый простой вариант:

```powershell
python -m http.server 8080
```

Или просто:

```powershell
.\start-server.ps1
```

После этого открой:

```text
http://localhost:8080
```

Скрипт `start-server.ps1` сам попробует сначала `python`, потом `py`.

## Как использовать

1. Вставь свой OpenAI API key.
2. При необходимости поменяй модель. По умолчанию стоит `gpt-5-mini`.
3. Выбери профиль: `profile-short.txt` или `profile-long.txt`.
4. Вставь HTML страницы.
5. Выбери режим `CSS` или `XPath`.
6. Укажи селектор и нажми `Extract`.
7. Проверь найденные блоки и итоговый текст.
8. Нажми `Analyze`.

## Замечания

- Это локальный V1 без бэкенда, поэтому API key вводится прямо в интерфейсе.
- Ключ нигде специально не сохраняется, но страница все равно не предназначена для публичного размещения.
- `API key`, модель и последний селектор сохраняются в `localStorage` браузера.
- В Chromium-браузерах можно выбрать папку для результатов, и JSON будет сохраняться туда автоматически с именем по текущей дате и времени.
- Профили читаются из файлов `profile-short.txt` и `profile-long.txt`, поэтому страницу лучше открывать через локальный HTTP server, а не просто двойным кликом по `index.html`.
- Промпт и JSON-схему результата можно менять прямо в интерфейсе.
