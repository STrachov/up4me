High-signal positives:
- client names concrete document types
- client names exact fields to extract
- output format is explicit
- sample files exist
- business workflow is clear
- success criteria are concrete
- there is mention of accuracy improvement, reconciliation, validation, confidence checks, or review flows
- client wants implementation of a real workflow, not a toy demo

Medium-signal positives:
- OCR is mentioned, but the workflow is still somewhat vague
- extraction is relevant, but document types are not yet fully specified
- project involves backend workflow and structured outputs, but scope needs clarification

Weak or low-signal fits:
- generic scraping
- generic web development

Red flags:
- vague "need OCR expert urgently" with no documents, no sample files, and no output requirements
- unrealistic scope relative to budget
- client expects support for all document types or all languages immediately
- no distinction between OCR, extraction, and manual data entry
- no stated workflow, no downstream use, no business context
- asks for a full production OCR SaaS on a tiny budget
- buzzword-heavy posting with no concrete extraction task
- contract reads like hidden full-time employment while posted as freelance

Evaluation lens:
When analyzing a job, decide:
1) Is this genuinely aligned with OCR / Document AI positioning?
2) Is the scope commercially realistic?
3) Is the project shape suitable for a freelance engagement?
4) Does the budget appear plausible for the complexity?
5) Is there enough specificity to justify applying now, or should this be a cautious maybe?

Scoring guidance:
- 80-100:
  direct OCR / Document AI fit, clear workflow, realistic scope, credible commercial setup
- 60-79:
  relevant fit, but some ambiguity on scope, docs, budget, or workflow detail
- 40-59:
  adjacent fit or technically possible, but weak positioning match or too much uncertainty
- 0-39:
  weak fit, commodity work, bad scope/budget, or poor commercial setup

Recommendation guidance:
- apply_now:
  strong fit, realistic scope, enough detail to justify engagement
- maybe:
  possible fit, but important unknowns remain
- skip:
  weak fit, poor economics, or too many structural red flags

Niche match guidance:
